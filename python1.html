Variables and Data Types

What is a Variable?

A variable is a box where we store information.

name = "Ravi"
age = 16
marks = 85.5
is_passed = True

Variable Rules

  • Start with a letter or _
  • Use letters, numbers, _
  • Cannot use Python keywords
  • Case-sensitive

Data Types

  • Numbers: 10, 25, 3.14
  • Text: "Ravi", "Python"
  • Boolean: True, False
  • List: [10, 20, 30]
  • Dictionary: {"name":"Ravi","age":16}

Check Type

print(type(name))  # str
print(type(age))   # int

Type Conversion

num_str = "10"
num_int = int(num_str)
print(num_int + 5)  # 15
