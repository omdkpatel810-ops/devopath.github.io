Python Operators

Arithmetic

print(5 + 2)  # 7
print(5 - 2)  # 3
print(5 * 2)  # 10
print(5 / 2)  # 2.5
print(5 % 2)  # 1
print(5 ** 2) # 25

Comparison

print(5 > 2)  # True
print(5 == 2) # False

Logical

print(True and False) # False
print(True or False)  # True
print(not True)       # False

Assignment

x = 5
x += 2  # x = x + 2
print(x) # 7
