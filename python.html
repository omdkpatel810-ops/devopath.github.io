Learn Python

What is Programming?

Programming is giving instructions to a computer to do tasks. We write these instructions in a language the computer can understand.

Why Python?

  • Simple and easy to read
  • Used in websites, data, games, and automation
  • Big community and lots of ready-made tools

Installing Python

  1. Go to python.org and download Python.
  2. Run the installer and check “Add Python to PATH”.
  3. Check installation: python --version

Choosing an IDE

An IDE helps you write, run, and test code:

  • VS Code (lightweight)
  • PyCharm (advanced)
  • Jupyter Notebook (data)
  • IDLE (comes with Python)

First Example

name = input("Enter your name: ")
print("Hello " + name)

This replaces "Hello World" with something meaningful.
